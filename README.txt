Warehouse Enterprise ERP

Run on Windows:
1. Install Python 3.11+
2. Open Command Prompt in this folder
3. Run: pip install -r requirements.txt
4. Run: python WH_ENTERPRISE_WEB.py

To access from iPhone:
1. Run the app on your PC
2. Find the PC IP address (for example 192.168.1.25)
3. Open Safari on iPhone and go to:
   http://192.168.1.25:8000

Default login:
admin / admin123

Other sample users:
requester / requester123
approver / approver123
buyer / buyer123
plant / plant123
receiver / receiver123

The app uses SQLite, local uploads, and the local assets folder, so it is portable as a folder.
